## 🚀 Cómo verlo en local

```bash
git clone https://github.com/dbarrientos-snap/Portafolio-dbarrientos-snap.git
cd Portafolio-dbarrientos-snap
python3 -m http.server 8000
```

Luego abre `http://localhost:8000` en el navegador.
